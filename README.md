# BSIM_SHA1
工程的实现采用verilog ,仿真工具采用questasim，仿真采用UVM方法学进行了简单的仿真，设计仅供参考。

目前此工程中的数据填充部分的位宽仅支持128bit,如果需要支持更高的速率，需要增加数据填充部分的带宽。
工程的仿真不仅对填充部分进行了自动化check，而且对于计算结果也进行了自动化check。
